# URL-Hashing-System
A complete URL hashing system that helps to track and make large urls short.

## Application Tech Stack
* Frontend technologies:
  * Javascript 6
  * Frontend framework Ejs
  * HTML, CSS, JS

* Backend technologies:
  * NodeJS 12
  * MongoDB

* Deployment
  * Docker & Docker compose
